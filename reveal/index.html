ECMAScript

It`s common practice now to abbreviate the standarts specification to simply ES X, although the X will be changing in 2016 (to a yearly release cadence).

ECMAScript 1

First version of the JavaScript language standart, was released in June 1997.

JavaScript connected in HTML as follows:

					
					<html>
					  <body>
					    <script language="JavaScript">
						    document.write("It`s JavaScript")
					    </script>
					  </body>	
					</html>

ECMAScript 2

Second version was released in June 1998. Was added editional changes to keep the specification fully aligned with ISO/IEC 16262 international standart.

ECMAScript 3

Third version was released in December 1999.

Added:

  • regular expressions
					
					var text = "Hello. I`m Javascript";
					  if (/I/i.test(text)) {
					    document.write("True");
					  } else {
					    document.write("False");
						}
					// True
  • do/while
					
					var result = "";
					var i = 0;
					do {
					 i += 1;
					 result += i + " ";
					}
					while (i > 0 && i < 5) 
					document.write(result);
					// 1 2 3 4 5
  • try/catch exception handling
    • 						
						try {
						throw 'myException'; // generates an exception
						} catch (error) {
						// statements to handle any exceptions
						 document.write(error);
						}
  • formatting for numeric output and other

ECMAScript 4

Work began on ECMAScript 4 in early 2000 but was abandoned due to political differences concerning language complexity.

Many features proposed to the Fourth Edition have been completely dropped; some were incorporated into 6th editional.

ECMAScript 5

ECMAScript 5 was reseased in December 2009, more than a dexade after ES3.

Added:

  • 'strict mode'
					
					'use strict';
					x = 3.14;  // This will couse an error because x is not declared
  • getters and setters
					
						var obj = {
						 get propName() {
						 // getter, the code execude on getting obj.propName
						 },
						 set propName(value) {
						 // setter, the code execude on setting obj.propName = value
						 }
						};
  • library support for JSON (stringify, parse)
    • 					
					console.log(JSON.stringify({x: 5, y: 6}));
					// expected output: "{'x': 5, 'y': 6}"

					var json = "{'x': 5, 'y': 6}";
					var obj = JSON.parse(json);

					console.log(obj.x);
					// expected output: 5
  • reserved words as property names
    (new, for, break, null, try etc.)
  • Array.prototype.methods

forEach() 
						
							var array1 = ['a', 'b', 'c'];
							array1.forEach(function(element) {
							 console.log(element);
							})
							// expected output: "a"
							// expected output: "b"
							// expected output: "c"
map() 
					
					var array = [1, 4, 9];
					var mapArray = array.map(function(num) {
						num * 2;
					});
					console.log(mapArray);
					// expected output: Array [2, 8, 18]
filter() 
					
					var words = ['spray', 'limit', 'exuberant', 'destruction'];
					var result = words.filter(function(word) {
					 word.length > 6;
					});
					console.log(result);
					// expected output: Array ["exuberant", "destruction"]
some() 
					
					var array = [1, 2, 3, 4, 5];
					// checks whether an element is even
					var even = function(element) {
					 element % 2 === 0
					};
					console.log(array.some(even));
					// expected output: true
every() 
					
					var isSmallNum = function(currentValue) {
						currentValue < 40
					};
					var array = [1, 30, 39, 29, 10, 13];
					console.log(array.every(isSmallNum));
					// expected output: true
indexOf() / lastIndexOf() 
					
					var beasts = ['ant', 'bison', 'camel', 'duck', 'bison'];
					console.log(beasts.indexOf('bison'));
					// expected output: 1
						
					console.log(beasts.lastIndexOf('bison'));
					// expected output: 4

ECMAScript 6 / ECMAScript 2015

The 6th edition, initially known as ECMAScript 6 then and later renamed ti ECMAScript 2015, was finalized in June 2015.

Added new features:

  • arrow function expression

ES5

						
							function greetings (name) {
							 return 'hello ' + name;
							}

ES6

						
							const greetings = (name) => {
							 return `hello ${name}`;
							}
						
					
						
							const greetings = name => `hello ${name}`;
  • const keyword for constant local declarations
    • 						
							// ES5
							var word = 'Hello';

							// ES6
							const word = 'Hello';
  • let keyword for local declarations
    • 						
							// ES5
							var word = 'Hello';

							// ES6
							let word = 'Hello';
  • Promises

ES5

							
								function isGreater (a, b, cb) {
								var greater = false
								if(a > b) {
								 greater = true
								 }
								 cb(greater)
								}
								isGreater(1, 2, function (result) {
								if(result) {
								 console.log('greater');
								} else {
								 console.log('smaller')
								 }
								})

ES6

							
							const isGreater = (a, b) => {
							return new Promise((res, rej) => {
							 if(a > b) {
							  res(true)
							  } else { rej(false) }
							})
							}
							isGreater(1, 2)
							.then(result => {
							 console.log('greater')
							})
							.catch(result => {
							 console.log('smaller')
							})
  • Exporting & Importing Modules

ES5

							
							var myModule = { x:1, y:function(){ 
							 console.log('This is ES5') 
							 }}
							module.exports = myModule;
							
						
							
							var myModule = require('./myModule')

ES6

							
							const myModule = { x: 1, y: () => { 
							 console.log('This is ES5') 
							 }}
							export default myModule;
							
						
							
							import myModule from './myModule';
  • Classes

ES5

							
							var Person = function(name) {
							 this.name = name;
							};
								
							Person.prototype.sayName = function() {
							 console.log('Person '+this.name+' said his name');
							};
								
							var john = new Person('John');
							john.sayName(); // Person John said his name

ES6

							
							class Person {
							 constructor(name) {
							 this.name = name;
							 }
								
							 sayName() {
							 console.log(`Person ${this.name} said his name`);
							 }
							}
				
							const john = new Person('John');
							john.sayName(); // Person John said his name

ECMAScript 7 / ECMAScript 2016

The 7th edition, officially known as ECMAScript 2016, was finalized in June 2016. The major standart language features include:

  • Array.includes() method

ES6

							
							let numbers = [1, 2, 3, 4];
							if(numbers.indexOf(2) !== -1) {
							 console.log('Array contains value');
							}

ES7

							
							let numbers = [1, 2, 3, 4];
							if(numbers.includes(2)) {
							 console.log('Array contains value');
							}
  • exponentiation operator for numbers

ES6

							
							let x = 3;
							let y = 2;
							console.log(Math.pow(x, y)); 
							// 9

ES7

							
							let squared = 3 ** 2;
							console.log(squared); 
							// 9
  • async / await
					
					async function firstAsync() {
					 let promise = new Promise((res, rej) => {
						setTimeout(() => res("Now it's done!"), 1000)
					 });
					
					 // wait until the promise returns us a value
					 let result = await promise; 
						
					 // "Now it's done!"
					 alert(result); 
					 }
					};
					firstAsync();

ECMAScript 8 / ECMAScript 2017

Was finalized in June 2017. Includes:

  • Object.values()
    • 						
						const countries = {
						 BR: 'Brazil',
						 DE: 'Germany',
						 RO: 'Romania',
						 PL: 'Poland'
						};
						
					
    						
						Object.values(countries); 
						// ['Brazil', 'Germany', 'Romania', 'Poland']
  • Object.entries()
    • 						
						const countries = {
						 BR: 'Brazil',
						 DE: 'Germany',
						 RO: 'Romania',
						 PL: 'Poland'
						};

						Object.entries(countries); 

						//[['BR', 'Brazil'], ['DE', 'Germany'], ['RO', 'Romania'], ['PL','Poland']]
  • Object.getOwnPropertyDescriptors()
    • 						
						const object1 = {
						 property1: 42,
						};
						const descriptors1 = Object.getOwnPropertyDescriptors(object1);
							
						console.log(descriptors1.property1.writable);
						// expected output: true
							
						console.log(descriptors1.property1.value);
						// expected output: 42

ECMAScript 9 / ECMAScript 2018

Was finalized in June 2018. New features include:

  • rest
					
					const foo = {
					 empanadas: 6,
					 milanesas: 2,
					 yerba: "1 Kg",
					};
					const { yerba, ...rest } = foo;
					console.log(yerba); // Prints "1 Kg"
					console.log(rest); // Prints {empanadas: 6, milanesas: 2}

We can use the rest operator at most once and it must be last.

  • spread
					
					const foo = {
					 empanadas: 6,
					 milanesas: 2,
					 yerba: "1 Kg",
					};
					const bar = { ...foo, empanadas: 12  };
					console.log(bar); // Prints 
					/**
					* {
					*   empanadas: 12,
					*   milanesas: 2,
					*   yerba: "1 Kg",
					**/

If we have conflicts with keys (empanadas), last one wins (empanadas : 12). This is useful to override default keys.

  • finally()
    • 						
						fetch(url)
						 .then()
						 .catch()
						 .finally(() => console.log(`I'm always called!`));
  • and additions ro RegExp
